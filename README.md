# FERI-TicTacToe

Tic-Tac-Toe game with AI opponent at Artificial Intelligence (slo. Umetna Inteligenca).

Implements minimax algorithm with depth limit and alpha-beta pruning.

### Features

- Tic-Tac-Toe Game
  - Multiplayer Mode
  - AI Mode
- Minimax
  - Depth Limit (Difficulty)
  - Alpha-Beta Pruning

### Setup

**Dependencies:**
- [Qt](https://www.qt.io/)
