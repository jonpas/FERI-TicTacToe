# FERI-TicTacToe

Tic-Tac-Toe game with AI opponent at Artificial Intelligence (slo. Umetna Inteligenca).

Implements minimax algorithm with depth option and optionally expands it with alpha-beta pruning.

### Features

- Tic-Tac-Toe Game
- AI Opponent
- Step Solving

### Setup

**Dependencies:**
- [Qt](https://www.qt.io/)
